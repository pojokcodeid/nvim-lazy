# Codeigniter 4 Snippets for Vscode

This Extensions provides the Codeigniter 4 snippets

## Requirements

```bash
CodeIgniter Version : 4.1.4
```

## Install

Launch Code's command palette

```bash
ext install adereksisusanto.codeigniter4-snippets
```
